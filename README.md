# Hydro

This is a work-in-progress compiler for my programming language, Hydro. It is written in Rust and uses Inkwell to wrap the LLVM compiler infrastructure. For parsing I use Chumsky, and error visualization uses Ariadne. The project is heavily inspired by my previous compiler attempt, [cblang](https://github.com/carsonetb/cblang-llvm), which was written in Python.
